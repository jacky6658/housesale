## 🚀 30 秒快速上手

### 1. 設定監控（對話）

打開 Telegram，對您的龍蝦說：

```
幫我找竹山鎮 5-8 萬的住宅，20 坪以上，透天優先
```

龍蝦會確認細節，然後自動開始監控 ✅

### 2. 收到通知（自動）

每當找到符合條件的物件，龍蝦會推送：

```
🏠 新物件通知！
竹山鎮透天厝 - 6,800元/月 | 28坪
AI 評分：85 分

回覆：
1️⃣ 確認聯繫
2️⃣ 稍後決定
3️⃣ 不感興趣
```

### 3. 一鍵操作

回覆 `1` → 龍蝦自動產生專業開發訊息 ✅
