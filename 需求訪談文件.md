## 💡 訪談技巧提醒

### 開放式提問
```
✅ 好問法：
"您目前找案源遇到最大的困難是什麼？"
→ 讓客戶自由發揮，了解真實痛點

❌ 壞問法：
"您是不是覺得找案源很浪費時間？"
→ 引導式問題，容易得到偏頗答案
```

### 需求深挖
```
客戶說：「我希望系統很好用」

追問：
"您說的『好用』具體指什麼？
 是操作簡單、還是功能強大、還是速度快？"

→ 把模糊需求變成可執行的功能
```

### 預算探測
```
客戶說：「預算 20 萬」

追問：
"這 20 萬是包含未來 6 個月的維護嗎？
 還是只是開發費用？"

→ 避免預算認知落差
```

### 風險管理
```
客戶說：「我要 100% 準確判斷屋主」

回應：
"目前 AI 技術極限是 75-85%，
 但我們可以設計『人工覆核』機制，
 確保您不會漏掉真正的屋主，
 也不會浪費時間聯繫仲介。"

→ 管理期待，避免過度承諾
```
