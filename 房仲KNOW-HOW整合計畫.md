## 🎯 精準抓「空屋」的關鍵特徵

### A. 屋主急租訊號（目前系統沒有）

**應該加入的判斷：**
1. **價格異常低**（低於區域行情 15% 以上）
   - 需求：竹山鎮歷史租金資料庫
   - 實作：爬取 591 過去 3 個月同區域行情
   - 評分：低於行情 15% → +30 分

2. **刊登時間長**（超過 30 天仍未租出）
   - 需求：追蹤物件刊登日期
   - 實作：資料庫記錄 first_seen_date
   - 評分：30-60 天 → +15 分；> 60 天 → +25 分

3. **文案關鍵字**
   ```
   急租 → +15 分
   可議價 → +10 分
   即可入住 → +10 分
   屋主誠意出租 → +15 分
   降價出租 → +20 分
   ```

4. **多次降價紀錄**
   - 需求：定期爬取比對價格變化
   - 實作：price_history 欄位（JSON 格式）
   - 評分：降價 1 次 → +10 分；2 次以上 → +20 分

5. **聯絡方式是手機**（不是 0800 或市話）
   - ✅ 已實作（+15 分）

### B. 空屋品質判斷（目前系統沒有）

**應該加入的判斷：**

1. **照片數量與品質**
   ```
   < 3 張 → -20 分（可疑，可能是仲介套圖）
   3-5 張 → 0 分（正常）
   > 5 張 → +10 分（屋主用心）
   ```

2. **照片真實性辨識**（進階功能）
   - 檢測浮水印（仲介 Logo → -50 分）
   - 樣品屋示意圖 vs 實拍（需圖片 AI）

3. **屋況描述完整度**
   ```
   包含以下資訊 → 每項 +5 分：
   - 家具配置說明
   - 裝潢年份
   - 採光方向
   - 樓層資訊
   - 附近生活機能
   ```

4. **提供實際地址**（而非僅區域）
   ```
   完整地址 → +15 分
   只寫「竹山鎮 XX 路」 → 0 分
   只寫「竹山鎮」 → -10 分
   ```

5. **聯絡時段說明**
   ```
   「僅晚上 7-9 點」 → +10 分（屋主上班）
   「隨時可聯絡」 → -5 分（可能仲介）
   提供 LINE ID / WeChat → -5 分（仲介常用）
   ```

## 🦞 房仲大腦範例程式碼

### 範例 1：價格異常偵測
```javascript
// data/zhushan-rent-history.json
const avgRent = {
  "竹山鎮_延平路_2房": 6500,
  "竹山鎮_集山路_2房": 6000,
  "竹山鎮_其他_2房": 5500
};

function checkPriceAnomaly(listing) {
  const area = extractArea(listing.location);
  const rooms = extractRooms(listing.title);
  const key = `竹山鎮_${area}_${rooms}房`;
  
  const marketPrice = avgRent[key] || 5500;
  const deviation = (listing.price - marketPrice) / marketPrice;
  
  if (deviation < -0.15) {
    return { 
      flag: 'URGENT', 
      score: 30, 
      reason: `低於行情 ${Math.abs(deviation*100).toFixed(0)}%`
    };
  }
  return { flag: 'NORMAL', score: 0 };
}
```

### 範例 2：地段評分自動化
```javascript
async function evaluateLocation(address) {
  const checkpoints = [
    { name: '竹山國小', lat: 23.7571, lng: 120.6769, weight: 15 },
    { name: '竹山傳統市場', lat: 23.7580, lng: 120.6775, weight: 15 },
    { name: '延平路商圈', lat: 23.7565, lng: 120.6780, weight: 20 }
  ];
  
  const coords = await geocode(address);
  let score = 0;
  
  for (const point of checkpoints) {
    const distance = calculateDistance(coords, point);
    if (distance < 0.5) {  // 500m 內
      score += point.weight;
    }
  }
  
  return score;
}
```

### 範例 3：仲介帳號黑名單
```javascript
// data/account-history.json
const accountDB = {
  "user_12345": { listings: 25, firstSeen: "2025-01", type: "broker" },
  "user_67890": { listings: 1, firstSeen: "2026-02", type: "owner" }
};

function checkAccountType(posterId) {
  const account = accountDB[posterId];
  if (!account) return 0;
  
  if (account.listings > 10) {
    return -30;  // 明顯仲介
  } else if (account.listings > 3) {
    return -10;  // 可能小仲介
  }
  return +10;  // 可能屋主
}
```
