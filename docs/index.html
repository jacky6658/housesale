常見問題 Q&A

❓ 系統需要什麼電腦？

A: 最低規格 Intel i3、8GB RAM、100GB 硬碟，一般文書機或筆電即可。

❓ 電腦關機會怎樣？

A: 系統會自動重啟，斷網期間的物件不會錯過（591 有歷史紀錄）。

❓ API Key 會被盜用嗎？

A: API Key 加密儲存，設定每月預算上限（如 $10），異常呼叫自動停止。

❓ 為什麼不用免費 LLM？

A: 免費模型準確度 <70%，商用 LLM 成本極低（月 <$5），錯誤推薦浪費業務員時間。

❓ 可以監控幾個區域？

A: 基礎版 1 個、進階版 3-5 個、完整版無限制。區域可細到「竹山鎮延平路」。

❓ 用量暴增怎麼辦？

A: 系統自動警報，可調整監控頻率、減少區域、或切換至 Gemini Flash（成本降 70%）。

❓ 業務員需要懂技術嗎？

A: 不需要！日常使用只需會看 Telegram 訊息，設定可用對話方式。

❓ 可以先試用嗎？

A: 可以！提供 7 天免費試用，我們會到府安裝測試版，滿意後再簽約。
